== Description ==

This plugin used show a especific post, page or posts of one or any categories

Supose you have an area in your home with highlights, you can use this widgets to put the main posts, and don't need to concern in copy text, links and other in de text-widget.

== Installation ==

1. Upload `rvpagepostswidget.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
